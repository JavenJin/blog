Computer Vision - Experiment 7 - Find and draw outline experiments

Experimental objectives and requirements

‚ÄÉ‚ÄÉUnderstand the basic principle of finding image contours; master the code writing method of finding contours using OpenCV; master the code writing method of drawing contours using OpenCV.

Experiment content

‚ÄÉ‚ÄÉ (i) New project.

‚ÄÉ‚ÄÉ (ii) Configuring OpenCV in VS2015.

‚ÄÉ‚ÄÉ (iii) Using the findContours function in OpenCV to implement find contours;

‚ÄÉ‚ÄÉ (iv) Use the drawContours function in OpenCV to implement draw contours.

Experimental instruments, equipment

‚ÄÉ‚ÄÉA computer with Windows 7 operating system and Visual Studio 2015 installed

Experimental principle

‚ÄÉ‚ÄÉ (i) Although an edge detection algorithm like Canny can detect pixels at the boundary of the contours based on the difference between pixels, it does not take the contours as a whole. So, this experiment is to assemble these edge pixels into a contour.

‚ÄÉ‚ÄÉ (ii) A contour generally corresponds to a series of points, which is a curve in the image. Its representation may vary depending on the situation. In OpenCV, the findContours function can be used to find contours from a binary image. drawContours function is used to draw external or internal contours in the image .

Experimental steps

‚ÄÉ‚ÄÉ (i) Create a Visual Studio 2015 console application;

‚ÄÉ‚ÄÉ (ii) Configure OpenCV in Visual Studio 2015;

‚ÄÉ‚ÄÉ (iii) Write code to implement findContours using the findContours function to implement find contours;

‚ÄÉ‚ÄÉ (iv) Write code to implement drawContours using the drawContours function.

Experimental notes

‚ÄÉ‚ÄÉ (i) The method of configuring OpenCV in VS after completing the installation of OpenCV;

‚ÄÉ‚ÄÉ (ii) The functions and usage of the findContours function;

‚ÄÉ‚ÄÉ (iii) The functions and usage of the drawContours function.

Computer Vision - Experiment 3 - Image linear filtering experiment

Experimental objectives and requirements

‚ÄÉ‚ÄÉ (i) To master the principle of box filtering of images and the programming implementation method through experiments;

‚ÄÉ‚ÄÉ (ii) To master the principle of mean filtering of images and the programmed implementation method through experiments;

‚ÄÉ‚ÄÉ (ii) To master the principle of Gaussian filtering of images and the programmed implementation method through experiments.

Experiment content

‚ÄÉ‚ÄÉ (i) Use the boxFilter function in OpenCV to implement box filtering.

‚ÄÉ‚ÄÉ (ii) Implementing mean filtering using the blur function in OpenCV;

‚ÄÉ‚ÄÉ (iii) Gaussian filtering using GaussianBlur function in OpenCV.

Experimental instruments, equipment

‚ÄÉ‚ÄÉA computer with Windows 7 operating system and Visual Studio 2015 installed

Experimental principle

‚ÄÉ‚ÄÉ (i) Image filtering, which refers to the suppression of noise in the target image under the condition of preserving as much as possible the detailed features of the image, is an indispensable operation in image pre-processing, and the effectiveness of its processing will directly affect the validity and reliability of subsequent image processing and analysis. The output value of each pixel of the linear filter is the weighted sum of some input pixels. Linear filters are easy to construct, well and easy to analyze from a frequency response perspective.

‚ÄÉ‚ÄÉ (ii) In OpenCV, the following three commonly used linear filter operations are provided, which are encapsulated in separate functions and are very easy to use: boxFilter function, mean filter blur function; Gaussian filter GaussianBlur function.

Experimental steps

‚ÄÉ‚ÄÉ (i) Create the Visual Studio 2015 console program;

‚ÄÉ‚ÄÉ (ii) Configure OpenCV in Visual Studio 2015;

‚ÄÉ‚ÄÉ (iii) Write code to implement box filtering using the boxFilter function;

‚ÄÉ‚ÄÉ (iv) Write code to implement mean filtering using the blur function;

‚ÄÉ‚ÄÉ (v) Write code to implement Gaussian filtering using the gaussianBlur function.

Experimental notes

‚ÄÉ‚ÄÉ (i) Configure OpenCV in VS after completing the installation of OpenCV;

‚ÄÉ‚ÄÉ (ii) The functions and usage of boxFilter function, blur function and gaussianBlur function.

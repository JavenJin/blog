Experimental principle

â€ƒâ€ƒ (i) In the field of image processing and computer vision, how to extract the required feature information from the current image is the key to image recognition. In many applications fieldsh need to detect straight lines or circles quickly and accurately, one of the very effective ways to solve the problem is the Haff transform.

â€ƒâ€ƒ (ii) Haff line transform is a method used to find straight lines. Before using Haff line transform, the image is first processed for edge detection, i.e., the direct input of Haff line transform can only be edge binary image.

â€ƒâ€ƒ (iii) In OpenCV, the HoughLines function can be used to call the standard Haff transform and the multiscale Haff transform. And the HoughLinesP function is used to invoke the cumulative probability Hough transform. The cumulative probability Hough transform is executed very efficiently, and we prefer to use the HoughLinesP function compared to the HoughLines function.

//-----------------------------------„ÄêShowHelpText( )ÂáΩÊï∞„Äë------------------------

	//ËæìÂá∫Ê¨¢Ëøé‰ø°ÊÅØÂíåOpenCVÁâàÊú¨
	printf("\n\n\t\t\tÈùûÂ∏∏ÊÑüË∞¢Ë¥≠‰π∞„ÄäOpenCV3ÁºñÁ®ãÂÖ•Èó®„Äã‰∏Ä‰π¶ÔºÅ\n");

	printf("\n\n\t\t\t   ÂΩìÂâç‰ΩøÁî®ÁöÑOpenCVÁâàÊú¨‰∏∫Ôºö" CV_VERSION );

//-------------------------------„Äêmain( )ÂáΩÊï∞„Äë-----------------------------------------
//          ÊèèËø∞ÔºöÊéßÂà∂Âè∞Â∫îÁî®Á®ãÂ∫èÁöÑÂÖ•Âè£ÂáΩÊï∞ÔºåÊàë‰ª¨ÁöÑÁ®ãÂ∫è‰ªéËøôÈáåÂºÄÂßãÊâßË°å
//-------------------------------------------------------------------------------------
int main( )
{

	//„Äê1„Äë‰ª•ÁÅ∞Â∫¶Ê®°ÂºèËØªÂèñÂéüÂßãÂõæÂÉèÂπ∂ÊòæÁ§∫
	Mat srcImage = imread("feng.png", 0);
	if(!srcImage.data ) { printf("ËØªÂèñÂõæÁâáÈîôËØØÔºåËØ∑Á°ÆÂÆöÁõÆÂΩï‰∏ãÊòØÂê¶ÊúâimreadÂáΩÊï∞ÊåáÂÆöÂõæÁâáÂ≠òÂú®~ÔºÅ \n"); return false; } 
	imshow("ÂéüÂßãÂõæÂÉè" , srcImage);   

	ShowHelpText();

	//„Äê2„ÄëÂ∞ÜËæìÂÖ•ÂõæÂÉèÂª∂Êâ©Âà∞ÊúÄ‰Ω≥ÁöÑÂ∞∫ÂØ∏ÔºåËæπÁïåÁî®0Ë°•ÂÖÖ
	int m = getOptimalDFTSize( srcImage.rows );
	int n = getOptimalDFTSize( srcImage.cols ); 
	//Â∞ÜÊ∑ªÂä†ÁöÑÂÉèÁ¥†ÂàùÂßãÂåñ‰∏∫0.
	Mat padded;  
	copyMakeBorder(srcImage, padded, 0, m - srcImage.rows, 0, n - srcImage.cols, BORDER_CONSTANT, Scalar::all(0));

	//„Äê3„Äë‰∏∫ÂÇÖÁ´ãÂè∂ÂèòÊç¢ÁöÑÁªìÊûú(ÂÆûÈÉ®ÂíåËôöÈÉ®)ÂàÜÈÖçÂ≠òÂÇ®Á©∫Èó¥„ÄÇ
	//Â∞ÜplanesÊï∞ÁªÑÁªÑÂêàÂêàÂπ∂Êàê‰∏Ä‰∏™Â§öÈÄöÈÅìÁöÑÊï∞ÁªÑcomplexI
	Mat planes[] = {Mat_<float>(padded), Mat::zeros(padded.size(), CV_32F)};
	Mat complexI;
	merge(planes, 2, complexI);         

	//„Äê4„ÄëËøõË°åÂ∞±Âú∞Á¶ªÊï£ÂÇÖÈáåÂè∂ÂèòÊç¢
	dft(complexI, complexI);           

	//„Äê5„ÄëÂ∞ÜÂ§çÊï∞ËΩ¨Êç¢‰∏∫ÂπÖÂÄºÔºåÂç≥=> log(1 + sqrt(Re(DFT(I))^2 + Im(DFT(I))^2))
	split(complexI, planes); // Â∞ÜÂ§öÈÄöÈÅìÊï∞ÁªÑcomplexIÂàÜÁ¶ªÊàêÂá†‰∏™ÂçïÈÄöÈÅìÊï∞ÁªÑÔºåplanes[0] = Re(DFT(I), planes[1] = Im(DFT(I))
	//void magnitude(InputArray x, InputArray y, OutputArray magnitude);
	magnitude(planes[0], planes[1], planes[0]);// planes[0] = magnitude  
	Mat magnitudeImage = planes[0];

	magnitudeImage += Scalar::all(1);

	//„Äê7„ÄëÂâ™ÂàáÂíåÈáçÂàÜÂ∏ÉÂπÖÂ∫¶ÂõæË±°Èôê
	//Ëã•ÊúâÂ•áÊï∞Ë°åÊàñÂ•áÊï∞ÂàóÔºåËøõË°åÈ¢ëË∞±Ë£ÅÂâ™      
	magnitudeImage = magnitudeImage(Rect(0, 0, magnitudeImage.cols & -2, magnitudeImage.rows & -2));
	//ÈáçÊñ∞ÊéíÂàóÂÇÖÁ´ãÂè∂ÂõæÂÉè‰∏≠ÁöÑË±°ÈôêÔºå‰ΩøÂæóÂéüÁÇπ‰Ωç‰∫éÂõæÂÉè‰∏≠ÂøÉ  
	int cx = magnitudeImage.cols/2;
	int cy = magnitudeImage.rows/2;
	Mat q0(magnitudeImage, Rect(0, 0, cx, cy));   // ROIÂå∫ÂüüÁöÑÂ∑¶‰∏ä
	Mat q1(magnitudeImage, Rect(cx, 0, cx, cy));  // ROIÂå∫ÂüüÁöÑÂè≥‰∏ä
	Mat q2(magnitudeImage, Rect(0, cy, cx, cy));  // ROIÂå∫ÂüüÁöÑÂ∑¶‰∏ã
	Mat q3(magnitudeImage, Rect(cx, cy, cx, cy)); // ROIÂå∫ÂüüÁöÑÂè≥‰∏ã
	//‰∫§Êç¢Ë±°ÈôêÔºàÂ∑¶‰∏ä‰∏éÂè≥‰∏ãËøõË°å‰∫§Êç¢Ôºâ
	Mat tmp;

	//‰∫§Êç¢Ë±°ÈôêÔºàÂè≥‰∏ä‰∏éÂ∑¶‰∏ãËøõË°å‰∫§Êç¢Ôºâ
	q1.copyTo(tmp);                 
	q2.copyTo(q1);
	tmp.copyTo(q2);

	//Ê≠§Âè•‰ª£Á†ÅÁöÑOpenCV2Áâà‰∏∫Ôºö
	//normalize(magnitudeImage, magnitudeImage, 0, 1, CV_MINMAX);

	normalize(magnitudeImage, magnitudeImage, 0, 1, NORM_MINMAX);
